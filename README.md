<p align="center">

## Project Content
In this project, we have 5 pages which are represent our activity if we want to get ticket via online. 5 pages are representing sign up page, home page, detail movie page, order option page and payment page. Amoung pages that we have, automaticly can be connecting each other in certain button.

## Pages Name
- signUp
- index
- details
- order
- payment


## How it work

You can run this code on your localhost, and you need
- visual studio code (https://code.visualstudio.com/).
- browser, example : (chrome, mozila, exploler, dll).

If you get all require, you can open index.html in your browser or open with terminal on visual studio code

or if you want to deploy the code and you wanna access website online, you can use netlify or your own domain and server.
